Classi Study Record Editor
This is an editor for Classi Study Records.

Outline: I found the normal editor for Classi Study Record annoying to use. So I created an editor that is useful for me.
'Tool of the self-proclaimed prestigious school, by the self-proclaimed prestigious school, for the self-proclaimed prestigious school'.

How to use: Add main.js code to a bookmark in your browser. Then open your Classi Study Record and open the bookmark.
This system doesn't work if yourschool limits runing JScode from bookmarklet.
